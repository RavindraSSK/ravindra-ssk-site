# Ravindra SSK — Personal Website

Personal website of **Ravindra Siva Sai Kumar Medicharla** — M.S. in Artificial Intelligence student at Saint Louis University, focused on AI, Machine Learning, and Data Science.

🔗 Live site: **https://www.ravindrassk.com**

## About the Site

More than a portfolio — it's a personal hub that brings together my work, writing, and interests. It's a fast, SEO-optimized site with light/dark themes and custom branding.

## Sections

- **Home** — landing page and introduction.
- **About** — background, education, and journey into AI.
- **Portfolio** — projects, skills, and achievements.
- **Explore (Blog)** — articles and writing on AI/tech and other interests (updated over time).
- **Contact** — get in touch via a working contact form.

## Tech Stack

- **Framework:** Next.js 16 (App Router) + React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Analytics & Hosting:** Vercel (Web Analytics + Speed Insights)
- **SEO:** sitemap, robots, manifest, OpenGraph + Twitter cards
